# 🎮 Stone Paper Scissors Game (Python)

This is a simple **Stone-Paper-Scissors game** built using Python. It allows a user to play against the computer in a loop until they choose to exit.

---

## 🚀 Features

* User vs Computer gameplay
* Random computer choices using Python's `random` module
* Continuous gameplay (loop) until user exits
* Simple and beginner-friendly logic

---

## 🛠️ Technologies Used

* Python
* Random Module

---

## ▶️ How to Run the Project

1. Clone the repository from GitHub:

   ```
   git clone <your-repo-url>
   ```

2. Navigate to the project folder:

   ```
   cd your-folder-name
   ```

3. Run the Python file:

   ```
   python Stone_paper_scissors.py
   ```

---

## 🎯 How to Play

* Enter:

  * `1` → Stone
  * `0` → Paper
  * `-1` → Scissors

* The computer will randomly choose its move.

* The result (Win / Lose / Draw) will be displayed.

* You can continue playing until you type **no**.
